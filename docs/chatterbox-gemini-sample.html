Open Audio Judge Sample

Chatterbox TTS generation scored by the Gemini audio judge.

What This Demonstrates

This sample shows the TTS-first AutoJudge workflow: build or load TTS cases, synthesize audio with a local open model, send the generated audio plus target text to Gemini, and review an aggregate report with per-sample diagnostics, category averages, and multi-judge sampled scores.

1. Cases
JSONL cases with target text, turns, and metadata.
2. Generator
Chatterbox writes local audio files under runs/.
3. Judge
Gemini listens three times per sample and returns strict JSON scores.
4. Report
HTML and JSONL summarize category scores, individual samples, and provenance.

How To Run It

Use this command for the expanded category demo. Source GEMINI_API_KEY at runtime; do not commit it to the repository. 

oaj autojudge-local-tts \
  --model mlx-community/chatterbox-turbo-6bit \
  --cases examples/tts_multiturn_cases.jsonl \
  --judge-provider gemini \
  --judge-samples 3 \
  --limit 10 \
  --out runs/chatterbox-tts-autojudge-smoke

If the local console script is stale while developing from source, run the same command through the module entry point: 

PYTHONPATH=src:. .venv/bin/python -m open_audio_judge.cli autojudge-local-tts \
  --model mlx-community/chatterbox-turbo-6bit \
  --cases examples/tts_multiturn_cases.jsonl \
  --judge-provider gemini \
  --judge-samples 3 \
  --limit 10 \
  --out runs/chatterbox-tts-autojudge-smoke

How To Read The Report

Start with average score and label counts, then inspect score breakdowns by evaluation category, TTS slice, model, voice, and language. Use the top diagnostics to find repeated issue clusters such as pronunciation errors, artifact classes, or weak slices. Finally, open the per-sample rows to inspect the averaged score, the three underlying Gemini scores, target text hash, generated-audio provenance, and any researcher notes.

This page is a committed sample artifact. Real run outputs should stay under ignored runs/ directories unless the audio, text, and metadata are explicitly safe to publish.
