## Remaining Risks

- Gemini sample records are committed smoke snapshots, not a substitute for recurring live provider checks.
- ASR semantic heuristics are useful for mock calibration but should not be treated as a human-equivalent semantic metric.
- TTS open samples are human recordings used as development fixtures; production benchmark data still needs pinned dataset cards, licenses, checksums, and synthesis provenance.
- There is no CI workflow yet, so local test health depends on contributors running the documented commands.
